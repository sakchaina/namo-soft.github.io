ปณท (THP Core System)

ยกระดับศักยภาพการปฏิบัติงานด้วยระบบใหม่ที่เชื่อมโยงทุกมิติของการให้บริการลูกค้า การจัดการเครือข่าย และระบบหลังบ้านอย่างบูรณาการ

ภาพรวมหลักสูตรการฝึกอบรม

ครอบคลุมการทำงานของระบบ THP Core System ทั้ง 4 โมดูลหลัก มุ่งเน้นความเข้าใจในกระบวนการทำงานแบบ End-to-End

เจาะลึก 4 โมดูลหลัก

รายละเอียดและหัวข้อย่อยของแต่ละระบบที่ผู้เข้ารับการอบรมจะได้เรียนรู้เพื่อนำไปปฏิบัติงานจริง

OMNICHANNEL

บริหารจัดการเครือข่าย

  • ✓ ภาพรวมระบบและการลงชื่อเข้าใช้งาน
  • ✓ การจัดการ Workflow Config
  • ✓ บริหารข้อมูลเครือข่าย ปณอ./ปณร./EMS Point
  • ✓ การจัดการรายการพัสดุและครุภัณฑ์
  • ✓ การกำหนดค่าตอบแทนและการประเมินคุณภาพ

BOM ส่วนกลาง

Back Office Management

  • ✓ กระบวนการจัดการผู้ใช้งานและเครือข่าย
  • ✓ การจัดการข้อมูลคู่ค้าและสินค้า
  • ✓ การจัดการซื้อ/ขายสด POS/ขายเชื่อ
  • ✓ การจัดการเงินสด เช็ค และเงินสดย่อย
  • ✓ การกระทบยอดและการเชื่อมต่อระบบ ERP (SAP)

PRICING & MARKETING

Price & Promotion

  • ✓ การจัดการราคาและข้อมูลสินค้าใน/ระหว่างประเทศ
  • ✓ การจัดการกลุ่มลูกค้าและโปรโมชั่น
  • ✓ การจัดการค่าตอบแทนคู่ค้า
  • ✓ การจัดการข้อกำหนดขนส่งและ Surcharge

CMP / OMS / QUEUE

ระบบ Customer Portal

  • ✓ CMP: การลงทะเบียนสมาชิกและสร้างใบจ่าหน้า
  • ✓ OMS: การจัดการ Order และคลังสินค้า
  • ✓ QUEUE: การจัดการระบบคิวและจองคิวออนไลน์
  • ✓ Admin: การจัดการ Dashboard และ Permission

แผนผังความเชื่อมโยงระบบ (System Flow)

การทำงานแบบบูรณาการ ไหลลื่นจากจุดสัมผัสลูกค้า (Customer Touchpoint) ไปจนถึงการจัดการหลังบ้าน (Back Office)

กลุ่มเป้าหมาย

บุคลากรที่เกี่ยวข้องกับการปฏิบัติการไปรษณีย์, เจ้าหน้าที่ระบบ IT, ผู้ดูแลเครือข่าย ปณอ./ปณร., และเจ้าหน้าที่ฝ่ายบัญชี/การเงินที่ต้องใช้งานระบบ ERP

รายละเอียดการปฏิบัติ

ผู้เข้าอบรมจะได้ลงมือปฏิบัติจริงบนระบบจำลอง (Test Environment) ครอบคลุมตั้งแต่การตั้งค่า Workflow ไปจนถึงการกระทบยอดสิ้นวัน

บทสรุปผู้บริหาร

การบูรณาการระบบ THP Core System ทั้ง 4 ส่วน จะช่วยลดความซ้ำซ้อน เพิ่มความแม่นยำในการคิดราคาและโปรโมชั่น และยกระดับประสบการณ์ลูกค้าผ่าน Portal ที่ทันสมัย
